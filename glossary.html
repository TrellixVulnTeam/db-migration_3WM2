ACeDB
The current, legacy database technology.
ARN
Amazon Resource Name
AWS
Amazon Web Services - the umbrella term for all of the Amazon services described in this project.
aws cli
The Amazon Command Line Interface
azanium
The name of this software package.
azanium admin
The azanium sub-command used to administer WormBase AWS accounts for the database migration.
azanium run
The azanium sub-command used on an AWS EC2 instance in order to perform the database migration.
circusd
A daemon that runs the circus process watcher.
CLI
Command Line Interface.
client commands
Commands issued from a host machine outside of the AWS network.
Datomic
The new WormBase database technology.
EC2
Amazon’s Elastic Compute service.
IAM
Amazon Identity and Access Management
instance commands
Commands issued on the EC2 instance provisioned for the build.
pseudoace
The name of the software package used to migrate to the Datomic database.
S3
Amazon S3 storage service.
screen
The most infamous program used to preserve sessions on remote hosts.
Slack
The messaging service used for WormBase inter-team communications.
tmux
A newer program used to preserve sessions on remote hosts with less irritating key bindings.
